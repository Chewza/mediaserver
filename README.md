# plexms-config #

### Description ###

* my plex media server configuration for ubuntu

### Usage ###

```bash
git clone git@github.com:klutchell/plexms-config.git ~/plexms
sudo ~/plexms/bin/install
sudo ~/plexms/bin/configure
~/plexms/bin/pull
~/plexms/bin/up
```
